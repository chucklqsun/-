# 《程序员的算法趣题》
python版本的解法，使用jupyter notebook，仅用于学术讨论。

本书为那些学习过排序、搜索等知名算法的程序员准备了69道数学谜题的解题程序。

### 如何在本地使用
```
pip3 install --upgrade pip
pip3 install jupyter
jupyter notebook
```
